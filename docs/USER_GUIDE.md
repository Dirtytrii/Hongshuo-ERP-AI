# 用户操作指南

## 登录与角色

系统通过右上角切换用户角色模拟不同权限：

- **管理员**：全部功能、权限配置、重置数据。
- **项目经理**：项目、物料审批、财务查看、AI。
- **财务**：财务录入与审批、物料查看、AI。
- **录入员**：物料录入与出库申请、AI。

## 主要功能

- **仪表盘**：在建项目数、出库量、库存预警、待审批数量及最近动态。
- **项目管理**：项目列表的增删改、合同金额与进度。
- **物料仓库**：实时库存、入库登记、出库申请；项目经理/管理员可审批出库。
- **物料管理**：物料主数据维护与预警阈值。
- **财务收支**：收入/支出记录及审批。录入**收入**时，可选择「计入回款计划节点」；该节点的已收金额将按关联收入自动汇总，便于按节点对账。
- **审批中心**：统一查看财务单、变更单、报销单、借款单、还款单待办，支持按类型/关键字筛选并跳转到对应单据页面处理。
- **集成中心**：配置钉钉机器人通知、移动端接口开关、Web 回跳地址与通知模板，并可直接发送测试消息。
- **操作日志**：管理员可查看系统操作记录。
- **AI 决策室**：输入问题，基于当前数据做分析与建议。
- **消息中心**（铃铛图标）：待审批出库、财务待审批、低库存预警提醒。

## 导出与备份

- 在项目列表、财务记录、物料管理页面可点击 **导出 Excel** 下载当前数据。
- 管理员可使用 **重置数据** 清空并重新初始化（谨慎使用）。

## 常见问题

- **后端连接失败**：确认后端已启动（默认 8080 端口）。
- **AI 无响应**：检查 `.env` 中 `DEEPSEEK_API_KEY` 是否配置且有效（AI 决策室使用 DeepSeek API）。
- **钉钉没有收到通知**：到「集成中心」确认已开启钉钉通知、Webhook 可用，且通知模板未破坏 `%s` 占位符格式。
