# 宏硕建设 ERP 系统 - 项目介绍（业务与管理版）

> 本文档面向业务部门与管理层，介绍系统定位、核心价值、已实现功能及使用收益，便于汇报与推广。

---

## 一、系统定位

**宏硕建设 ERP** 是专为建筑/工程公司打造的**轻量级业务管理系统**，围绕「**抓回款、控成本、防漏账、清责任**」四大诉求，覆盖项目、物料、财务与供应商的完整业务闭环，并支持按角色做数据隔离与权限控制，方便一线录入、管理层决策。

- **不是**：大而全的通用 ERP，也不替代钉钉/企业微信的日常沟通。
- **是**：以项目为中心、以回款与成本为主线的**业财一体化**管理平台，数据准确、口径统一、可追溯。

---

## 二、核心价值（一句话）

- **老板/管理层**：一眼看清「该催哪笔款、哪个项目超支、还欠供应商多少钱、谁对哪个项目负责」。
- **项目经理**：只看到自己负责的项目，录里程碑、看成本、配合回款计划，数据不被别人看到。
- **财务**：收支按类别规范录入，审批通过后自动回写项目成本与已收款，月底报表口径一致。
- **库管/录入**：物料出入库与项目挂钩，出库参与项目材料成本，与财务支出一起形成完整成本视图。

## 三、已实现功能概览

### 3.1 项目管理与责任到人（P0）

- **项目经理绑定**：项目必须指定系统内的项目经理（用户下拉选择，不可手输），责任清晰。
- **数据隔离**：项目经理登录后只能查看和操作自己负责的项目；高管/财务可看全部项目。
- **里程碑与进度**：按里程碑完成情况自动计算项目进度；支持**里程碑超期预警**（计划日已过未完成的节点在仪表盘与项目详情中标红提醒）。

### 3.2 财务规范与回款管理（P0）

- **支出类别标准化**：支出只能选预定义类别（如人工费、材料费、机械费、分包费、间接管理费），杜绝「买砖」「材料采购」等口径混乱，报表可统计。
- **回款计划与催款预警**：每个项目可维护多条回款节点（预付款、进度款、结算款、质保金等），含计划日期、计划金额、已收金额与状态；**仪表盘提供「近期待催款预警」**，未来 7/15 日内该收未收的项目与金额一目了然。**收入可计入某回款节点**：录入或审批收入时可选「计入回款计划节点」，节点已收金额将按关联收入自动汇总。
- **成本口径统一**：项目详情页分项展示——来自财务的材料成本、来自出库的材料金额、材料成本合计，并注明数据来源，财务与项目经理口径一致（材料成本 = 财务直发项目的材料支出 + 仓库领用出库折算金额）。

### 3.3 供应商与应付管理（P1）

- **客商档案**：统一维护供应商/分包商主数据（名称、联系人、银行信息等）。
- **业务关联**：入库、财务支出可关联到具体供应商，便于追溯。
- **应付/已付/欠款**：系统支持按供应商汇总应付、已付、欠款，生成可供签字付款前查阅的明细表。

### 3.4 预算与成本管控（P1）

- **项目预算**：建项目时可录入控制预算（总预算）；项目详情与列表展示预算与实际成本对比。
- **超支预警**：实际成本达预算 **80%** 时**黄灯预警**，达 **100%** 时**红灯并限制**继续对该项目做支出/出库（或需管理员特批），防止漏底。

### 3.5 变更与签证（P1）

- **变更/签证单**：支持录入签证事由、预估增加金额、甲方确认状态等；审批通过后可**自动追加到项目合同金额**，与现有合同额、回款计划衔接。
- **项目详情联动**：项目详情中可查看关联的变更单列表，便于结算与对账。

### 3.6 其他已具备能力

- **用户与角色**：登录认证、用户管理、角色与权限配置（如 admin / pm / finance / clerk），控制谁能看到哪些菜单与操作。
- **物料与库存**：实时库存、入库登记、出库申请与审批、低库存预警；出库与项目关联并参与项目材料成本统计。
- **仪表盘与报表**：仪表盘集中展示在建项目、待催款、超期里程碑、预算预警等；财务报表、库存报表、项目成本利润表支持筛选与导出。
- **AI 决策室（可选）**：集成大模型对项目、库存、财务数据进行自然语言分析与建议，便于管理层快速问数。
- **操作日志**：关键操作留痕，支持审计与问题追溯。

## 四、适用对象与典型场景

| 角色      | 典型使用场景                                                             |
| --------- | ------------------------------------------------------------------------ |
| 老板/高管 | 看仪表盘待催款与超支预警；查供应商欠款表；审阅项目利润与变更单。         |
| 项目经理  | 维护自己负责项目的里程碑与回款计划；查看项目成本构成与预算预警。         |
| 财务      | 录入收支并选规范类别；审批支出/收入；查看项目成本与财务报表。            |
| 库管/录入 | 维护物料与库存；做入库、出库并关联项目和供应商；配合财务与项目数据一致。 |

---

## 五、使用收益（可对外表述）

- **回款更清晰**：回款计划 + 待催款预警，避免「该收的款忘了催」。
- **成本有依据**：支出类别统一、材料成本 = 财务 + 出库，项目利润计算有据可查。
- **责任可追溯**：项目经理绑定项目、数据隔离，谁的项目谁负责；变更单留痕，结算有据。
- **欠款可回答**：供应商应付/已付/欠款一目了然，付款决策有数据支撑。
- **超支可防控**：预算 + 80%/100% 预警与锁定，减少「项目做完了才发现亏了」。

## 六、后续可拓展方向（非承诺）

- 工程文件/图纸管理：当前建议用钉钉文档或企业微盘管理，系统内仅保留文档清单或链接。
- 更复杂的项目看板/甘特图：当前以里程碑与超期预警为主，甘特图视业务需要再排期。
- **红字冲销**：财务支出/收入支持红字冲销（冲销单走审批流程，审批通过后回减项目成本/已收款）；库存冲销待实现。
